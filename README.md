# 배포 링크

# 프로젝트의 실행 방법

1. root 경로에 .env 파일 생성하여 아래의 내용을 추가합니다.
   ```js
   REACT_APP_WAS=https://pre-onboarding-selection-task.shop
   ```
2. 프로젝트 패키지룰 설치 합니다
   npm install
3. 프로젝트를 실행합니다.
   npm start

- CSS
